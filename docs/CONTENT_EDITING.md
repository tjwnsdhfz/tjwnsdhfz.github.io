# 포트폴리오 지원처별 편집 가이드

공개 편집기: <https://tjwnsdhfz.github.io/edit/>

## 가장 빠른 편집 순서

1. `소개·지원방향`에서 `범용 자기소개 적용`을 누릅니다.
2. `포트폴리오 포지셔닝`을 지원 분야에 맞는 한 줄로 조정합니다.
3. `핵심 역량`에서 각 항목의 `업무 적용 방식`만 지원 공고의 언어에 맞춰 수정합니다.
4. `프로젝트`에서 관련성이 높은 작업을 노출하고 순서를 바꿉니다.
5. `현재 버전 JSON 내보내기`로 지원처별 파일을 보관합니다.
6. 확인이 끝난 뒤에만 `게시·배포`에서 GitHub 공개본을 갱신합니다.

## 범용 소개가 바꾸는 범위

`범용 자기소개 적용`은 포지셔닝, Hero, 프로필 소개, 핵심 역량의 적용 문구, 섹션 소개와 마지막 연락 문구를 바꿉니다. 프로젝트, 저장소, 경력, 수상, 수치, 자격과 연락처는 유지합니다.

## 저장 상태 구분

- 브라우저 초안: 현재 브라우저에만 저장됩니다. 공개 사이트에는 반영되지 않습니다.
- JSON 파일: 지원처별 버전을 로컬 파일로 보관하거나 다시 불러올 때 사용합니다.
- GitHub 공개본: `public/portfolio-content.json`을 갱신하고 GitHub Pages 재배포를 시작합니다.

기존 직무 맞춤 초안은 자동으로 불러오지 않습니다. 필요한 경우 편집기 상단의 `이전 직무 초안 불러오기`를 직접 선택하세요.
